# KIE.ai API Documentation

API documentation organized by **model/provider** for easy reference.

## Structure

```
kie-ai-docs/
└── models/
    ├── _general/       # General endpoints (auth, webhooks, credits)
    ├── elevenlabs/     # Audio (TTS, STT, sound effects)
    ├── flux/           # Flux Kontext image generation/editing
    ├── gpt4o/          # GPT-4o Image generation
    ├── grok/           # Grok image & video generation
    ├── kling/          # Kling 3.0 text-to-video
    ├── market/         # KIE Market API
    ├── midjourney/     # Midjourney image generation
    ├── nano-banana/    # Nano Banana 2 model
    ├── qwen/           # Qwen text-to-image
    ├── seedance/       # Seedance 2 image-to-video
    ├── seedream/       # Seedream image models (4.5, Pro, Lite)
    ├── sora/           # Sora 2 Pro video
    ├── suno/           # Suno music generation
    ├── tools/          # Upscale, remove background
    ├── upload/         # File upload (URL, base64, stream)
    ├── veo/            # Google Veo 3.1 video
    └── wan/            # Wan 2.7 video
```
